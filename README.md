# Maison Mono

**Тема проекта:** Онлайн-каталог капсульной одежды (модный минимализм).  
Проект демонстрирует многостраничный сайт на HTML5 и CSS3 с семантической разметкой, адаптивной вёрсткой и архитектурой CSS (reset/base/layout/components).

## Страницы
- Главная — `index.html`
- О студии — `pages/about.html`
- Коллекция (каталог) — `pages/collection.html`
- Контакты — `pages/contacts.html`

## Технологии
- HTML5 (семантические теги)
- CSS3 (Flexbox, Grid, адаптив)
- Без JavaScript, без CSS-фреймворков
